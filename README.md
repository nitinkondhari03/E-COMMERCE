## Run Locally

Clone the project

```bash
  git clone https://github.com/uzairansari11/native-order-609.git
```

Go to the project directory

```bash
  cd my-native-order-609
```

Install dependencies

```bash
  npm install
  npm i axios
  npm i react-redux
  npm i redux
  npm i thunk
  npm i json-server
  npm i chakra-ui
  npm i material-UI
  npm i material-icon
  npm i bcrypt
  npm i cors
  npm i dotenv
  npm i express
  npm i jsonwebtoken
  npm i mongoose
  npm i nodemon
```

Start the server

```bash
  npm run server
  npm start
```
